Art Institute App

Overview

Art Institute App is an iOS application built with SwiftUI that allows users to browse artwork information from the Art Institute of Chicago API. The app demonstrates API integration, JSON decoding, search functionality, navigation, and state management concepts covered in the SwiftUI Apprentice textbook.

Features

* Browse artwork information from a live REST API
* Search artworks by title
* View artwork details
* Pull-to-refresh support
* Loading state indicator
* Error handling with alerts
* SwiftUI NavigationStack navigation
